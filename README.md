## Installation

Frontend:

```bash
cd frontend
yarn
yarn dev
```

Backend:

1. Install Postgres database
2. setup .env file in backend folder

```bash
cd backend
yarn
yarn start
```

Hardhat:

```bash
cd hardhat
yarn
```
